

# 装甲突击：坦克战争 (Armored Assault: Tank Warfare)

![游戏Logo](https://placeholder-for-logo.com/logo.png)

## 📜 游戏简介

**装甲突击**是一款融合战略与动作的现代坦克战斗游戏。在这片硝烟弥漫的战场上，您将指挥精良的装甲部队，与敌军展开激烈对抗，保卫您的基地不被摧毁。游戏完美结合了经典坦克大战的怀旧元素与创新的现代游戏机制，为玩家带来震撼的战斗体验。

无论您选择单枪匹马深入敌后，与好友并肩作战，还是通过网络与远方的战友联手，都能在这片战火纷飞的土地上谱写属于您的传奇战史！准备好驾驶您的钢铁巨兽，在战场上叱咤风云了吗？

## ✨ 游戏特色

- **多样化游戏模式**：
  - 刺激的单人战役模式
  - 热血沸腾的双人本地合作模式
  - 突破地域限制的远程联机对战

- **丰富的坦克阵容**：
  - **轻型坦克**：高速度（⭐⭐⭐⭐）、快速攻击（⭐⭐⭐⭐）、装甲薄弱（⭐）
  - **标准坦克**：均衡性能（⭐⭐⭐）、适中火力（⭐⭐⭐）、中等防御（⭐⭐）
  - **重型坦克**：强大火力（⭐⭐⭐⭐⭐）、厚重装甲（⭐⭐⭐⭐⭐）、缓慢移动（⭐⭐）

- **多层次敌军系统**：
  - **基础坦克**：数量众多但实力较弱的常规敌军
  - **精英坦克**：配备特殊武器的危险对手
  - **终结者坦克**：装甲厚重、火力强大的终极挑战

- **战略性地形元素**：
  - 可破坏的砖墙提供临时掩护
  - 坚不可摧的钢铁墙改变战场布局
  - 危险的水域限制行动范围
  - 提供隐蔽的茂密草丛

- **强力增益道具系统**：
  - 攻击增强：提升火力50%，持续10秒
  - 生命恢复：立即恢复生命值
  - 无敌保护：短时间内免疫所有伤害
  - 炸弹：放置后爆炸，造成范围伤害
  - 护盾：提供额外防护层
  - 速度提升：大幅提高移动速度

- **精心设计的关卡挑战**：
  - 五大主题战场，每个战场都有独特的地形布局和敌军配置
  - 难度递增的关卡设计，考验玩家的战术应变能力
  - 惊险刺激的Boss战，需要灵活运用战术才能取胜

- **完善的游戏体验**：
  - 直观的游戏界面
  - 动态音效与背景音乐
  - 流畅的游戏控制
  - 便捷的存档系统

## 🎮 操作指南

### 单人游戏(1P)控制：
- **W, A, S, D** 或 **↑, ↓, ←, →** - 控制坦克移动
- **空格键** - 发射子弹
- **E键** - 使用炸弹
- **ESC** - 暂停游戏

### 双人游戏控制：
| 玩家 | 向上 | 向下 | 向左 | 向右 | 射击 | 炸弹 | 暂停 |
|------|------|------|------|------|------|------|------|
| 玩家1 | W | S | A | D | 空格 | E | ESC |
| 玩家2 | ↑ | ↓ | ← | → | 回车 | J | P |

### 游戏功能键：
- **F5** - 快速保存游戏
- **F9** - 快速加载游戏
- **M** - 静音/取消静音
- **Tab** - 显示计分板（联机模式）

## 💡 战略提示

- 不同类型的坦克适合不同的战斗风格，轻型坦克适合快速突袭，重型坦克适合正面对抗
- 合理利用地形优势：在草丛中设伏，利用墙壁阻挡敌人视线
- 水域会对坦克造成持续伤害，除非拥有无敌状态或护盾保护
- 精英和Boss敌军会掉落更好的道具，但也更难对付，需要谨慎应对
- 在双人模式中，一名玩家可以吸引火力，另一名玩家可以侧翼攻击
- 远程联机对战时，注意网络延迟可能带来的影响，提前预判敌人动向

## 🚀 安装与运行

1. 确保您的系统已安装Java 11或更高版本
2. 从官方网站下载最新版本的游戏安装包
3. 解压缩下载的文件到您选择的目录
4. 运行以下命令启动游戏：
   ```
   java --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml,javafx.graphics,javafx.media -jar ArmoredAssault.jar
   ```
5. 或者直接双击启动器文件（Windows系统下为`ArmoredAssault.exe`）
